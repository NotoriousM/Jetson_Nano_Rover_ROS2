## 5.1 Модель велосипеда

Кинематическая модель велосипеда — стандарт для одноосных роботов с управлением Аккермана:

```
Состояние:  q = [x, y, ψ]ᵀ
Управление: u = [v, δ]ᵀ

Уравнения:
  ẋ  = v · cos(ψ)
  ẏ  = v · sin(ψ)
  ψ̇  = v · tan(δ) / L

где:
  x, y  — позиция центра масс (м)
  ψ     — рыскание (рад), 0 = вперёд
  v     — линейная скорость (м/с)
  δ     — угол руля (рад), >0 = вправо
  L     — продольная база = a + b = 0.807 м
```

## 5.2 ICR — мгновенный центр вращения

```
Вид сверху:

                  δ (угол руля)
      ┌───────────────────────────────────────────────────┐
      │  FL             FR                                │
      │  ←α_L    base  →α_R                               │
      │                                                   │
      │  ML (нет серво) MR                                │
      │                                                   │
      │  ←β_L          →β_R                               │
      │  RL             RR                                │
      └───────────────────────────────────────────────────┘
                │
                │  W/2
                │
               ICR (0, R)  — в системе координат ровера
```

```
R = a / tan(δ)   — радиус ICR относительно задней оси

При δ → 0:  R → ∞  (прямолинейное движение)
При δ = δ_max = 35°:  R_min = a/tan(35°) = 0.4035/0.7002 ≈ 0.576 м
```

## 5.3 Углы рулевых суставов

Все четыре рулевых колеса должны смотреть на один ICR:

```
α_L  = atan(a / (R + W/2))    # угол FL (рад → ° для серво)
α_R  = atan(a / (R − W/2))    # угол FR

β_L  = atan(b / (R + W/2))    # угол RL
β_R  = atan(b / (R − W/2))    # угол RR

# Публикуется в /joint_states:
[+α_L, +α_R, −β_L, −β_R]

# В градусах для серво (нейтраль = 90°):
servo_deg = initial_pos_servo_deg + degrees(α_L)   # = 90 + α_L°
```

**Числовой пример** при δ = 15° (v = 0.5 м/с):
```
R   = 0.4035 / tan(15°) = 0.4035 / 0.268 = 1.505 м
α_L = atan(0.4035 / (1.505 + 0.390)) = atan(0.213) = 12.0°
α_R = atan(0.4035 / (1.505 − 0.390)) = atan(0.362) = 19.9°
β_L = atan(0.4035 / (1.505 + 0.390)) = 12.0°
β_R = atan(0.4035 / (1.505 − 0.390)) = 19.9°
```

## 5.4 Скорости 6 колёс

```
Координаты колёс в системе ровера (X=вперёд, Y=влево):

  Колесо   x_off (м)   y_off (м)
  FL       +0.4035     +0.3895
  ML        0.0        +0.3895
  RL       -0.4035     +0.3895
  FR       +0.4035     -0.3895
  MR        0.0        -0.3895
  RR       -0.4035     -0.3895

ICR находится в точке (0, R) в системе ровера.

Расстояние колеса до ICR:
  R_i = √((x_off_i)² + (y_off_i − R)²)

Линейная скорость колеса:
  v_i = v · R_i / |R|

Угловая скорость для ROS joint_states:
  ω_i = v_i / r_wheel · SIGN_i

  SIGN_i:
    Левые  (FL, ML, RL): +1
    Правые (FR, MR, RR): −1
    (правый рокер повёрнут на 180° в URDF)
```

## 5.5 Интегрирование RK4

```python
def rk4_step(state, v, delta, L, dt):
    """Один шаг интегрирования методом Рунге-Кутта 4-го порядка."""
    def f(s):
        x, y, psi = s
        return [v*cos(psi), v*sin(psi), v*tan(delta)/L]
    
    k1 = f(state)
    k2 = f([state[i] + 0.5*dt*k1[i] for i in range(3)])
    k3 = f([state[i] + 0.5*dt*k2[i] for i in range(3)])
    k4 = f([state[i] +     dt*k3[i] for i in range(3)])
    
    new_state = [state[i] + dt/6*(k1[i]+2*k2[i]+2*k3[i]+k4[i]) for i in range(3)]
    new_state[2] = atan2(sin(new_state[2]), cos(new_state[2]))  # нормализация ψ
    return new_state
```

RK4 даёт погрешность O(dt⁴), против O(dt) у метода Эйлера. При dt=0.02с это на 4 порядка точнее.

## 5.6 Ограничения системы

| Ограничение | Значение | Причина |
|-------------|---------|---------|
| `δ_max` | ±35° (0.611 рад) | Механика сервопривода |
| `R_min` | ~0.576 м | Из `δ_max` и `a=0.4035` |
| `v_max` | 2.0 м/с | Параметр конфигурации |
| `ω_max` | 3.47 рад/с | `v_max/R_min` |
| Средние колёса | угол = 90° (нейтраль) | Нет сервопривода |

**Минимальный радиус траектории лемнискаты:**
```
a_scale_min = 4·a / tan(δ_max) = 4·0.4035 / 0.7002 ≈ 2.31 м
(рекомендуется scale ≥ 2.8 м с запасом)
```
